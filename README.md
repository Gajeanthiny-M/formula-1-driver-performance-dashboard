# Formula 1 Driver Performance Dashboard (2018–Present)

## Overview
This project analyzes Formula 1 driver performance from 2018 to present using PostgreSQL, SQL, and Tableau.

## Tools Used
- PostgreSQL
- SQL
- Tableau
- CSV data extraction

## Key Metrics
- Racecraft Score
- Podium Finish Rate
- Wins
- Podiums
- Average Grid Position
- Average Finish Position

## Key Insights
- Max Verstappen led both wins and podium rate.
- Lance Stroll had the highest Racecraft Score, showing strong position recovery relative to starting position.
- Drivers who start near the front have fewer opportunities to gain positions, which affects Racecraft Score interpretation.
